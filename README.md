## 🔄 更新机制

- **定时更新**: 每月15日自动执行，同步上游变更。
- **手动触发**: 可在 Actions 页面手动运行此工作流，立即获取最新图标。
- **推送更新**: 向 `main` 分支的任何推送都会触发更新。

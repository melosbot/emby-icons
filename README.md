## 🚀 使用方法

直接复制上表中您需要的 CDN 链接，在支持该格式的客户端中进行配置即可。

### 示例

```txt
https://cdn.jsdelivr.net/gh/melosbot/emby-icons@main/icons/baiitang_yuan_tubiao.json
```

所有图标图片资源托管于 [`icons/assets`](icons/assets) 目录。

- **定时更新**: 每日自动执行，同步上游变更。
